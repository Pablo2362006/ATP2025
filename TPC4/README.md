Este código representa o TPC4 e consiste numa aplicação para a gestão de um sistema de salas de cinema. O programa utiliza uma estrutura de dados onde cada sala é armazenada como uma lista contendo o número total de lugares, os lugares já vendidos e o nome do filme em exibição. Através de um menu interativo, o utilizador pode consultar o cartaz atual com os filmes e as respetivas salas, bem como verificar a disponibilidade de lugares específicos para uma determinada sessão. O código permite ainda a compra de bilhetes, atualizando automaticamente a lista de lugares ocupados, e oferece funções para listar a quantidade de lugares disponíveis em cada sala. Por fim, a aplicação inclui funcionalidades para gerir a infraestrutura do cinema, permitindo verificar se uma sala já existe no sistema e adicionar novas salas com filmes e capacidades personalizadas.
