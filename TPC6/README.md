Este resumo do TPC6 integra o motor de cálculo (meteorologia.py), a interface de utilizador (appmeteorologia.py) e a base de dados real (meteorologia.txt).
O projeto consiste num sistema completo de monitorização climatérica em Python, estruturado para processar e analisar registos históricos de temperatura e pluviosidade
O programa utiliza o ficheiro de texto externo meteorologia.txt como base de dados, onde cada linha armazena de forma estruturada a data, as temperaturas extremas e o índice de precipitação. Através do módulo de funções, o sistema consegue carregar estes dados brutos e transformá-los em informações úteis, permitindo ao utilizador, via um menu interativo, calcular médias e amplitudes térmicas, detetar a temperatura mínima mais baixa e identificar os dias de maior chuva. A lógica implementada permite ainda realizar análises estatísticas mais complexas, como a contagem de dias chuvosos e o cálculo do maior período seco consecutivo, baseando-se em limites personalizados introduzidos pelo utilizador. Finalmente, o sistema tira partido da biblioteca matplotlib para converter estes registos numéricos em gráficos visuais, facilitando a interpretação das tendências meteorológicas ao longo do tempo.
