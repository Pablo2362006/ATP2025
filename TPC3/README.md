Este código do TPC3 representa uma aplicação interativa focada na manipulação e análise de listas de números inteiros. Através de um menu principal, o programa permite ao utilizador gerir dados de forma dinâmica, oferecendo opções para criar listas tanto manualmente como de forma aleatória. Além da gestão básica, a aplicação executa operações matemáticas e estatísticas, como o cálculo da soma, da média aritmética e a identificação dos valores máximo e mínimo da lista. O sistema inclui ainda lógica para verificar se os elementos estão organizados por ordem crescente ou decrescente, bem como uma funcionalidade de pesquisa para localizar a posição de um número específico dentro da estrutura de dados.
