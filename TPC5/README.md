Este código do TPC5 consiste numa aplicação desenvolvida para a gestão de turmas e alunos. O programa utiliza uma estrutura de dados onde cada aluno é registado como um conjunto de informações que inclui o seu nome, um identificador (ID) único e um conjunto de notas referentes ao TPC, projeto e teste.
O utilizador pode criar uma nova turma do zero ou adicionar alunos individualmente, inserindo os seus dados pessoais e respetivas avaliações académicas. O sistema oferece ferramentas de consulta que permitem listar todos os membros de uma turma ou localizar um aluno específico através do seu número de identificação. Além da gestão em memória, o código inclui uma componente de persistência de dados, possibilitando guardar todas as informações da turma num ficheiro de texto externo e, posteriormente, carregar esses mesmos dados para continuar a trabalhar no programa. Toda a interação é feita através de um menu intuitivo que orienta o utilizador entre as tarefas de organização, visualização e armazenamento de dados escolares
